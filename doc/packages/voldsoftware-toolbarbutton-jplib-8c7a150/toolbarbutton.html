toolbarbutton

The toolbarbutton API provides a simple way to create toolbar buttons, which can perform an action when clicked.

Example

exports.main = function(options) {
  // create toolbarbutton
  var tbb = require("toolbarbutton").ToolbarButton({
    id: "TBB-TEST",
    label: "TBB TEST",
    onCommand: function () {
      tbb.destroy(); 
    }
  });

  if (options.loadReason == "install") {
    tbb.moveTo({
      toolbarID: "nav-bar",
      forceMove: false // only move from palette
    });
  }
};

API Reference

Classes

ToolbarButton

Module exports ToolbarButton constructor allowing users to create a toolbar button.

Constructors
ToolbarButton(options)

Creates a toolbarbutton.

options : Object

Options for the toolbarbutton, with the following parameters:

id : String

A id for the toolbar button, this should be namespaced.

label : String

A label for the toolbar button.

image : String

A image url for the toolbar button.

[ onCommand : Function ]

A option function that is invoked when the toolbar button is pressed.

[ panel : Panel ]

A optional panel.

Methods
destroy()

Removes the toolbar button from all open windows and no longer adds the toolbar button to new windows.

moveTo(options)

Moves the toolbar button on all open windows to the desired location.

options : Object

Options which describe the position to move the toolbar button to, with the following parameters:

toolbarID : String

The id of the toolbar which you want to add the toolbar button to.

Example toolbar IDs:

  • toolbar-menubar: The menu bar.
  • nav-bar: The navigation bar.
  • TabsToolbar: The tabs bar.
  • addon-bar: The addon bar.
insertbefore : String

The id of the element which the toolbar button should be inserted before.

forceMove : Boolean

If forceMove is false, then the move will only occur if the toolbar button is not already being used. If true, then the move will happen no matter where the toolbar button is.
